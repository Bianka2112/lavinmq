# AvalancheMQ

A message queue server that implements the AMQP 0-9-1 protocol. Written in [Crystal](https://crystal-lang.org/).

## Installation

```
make
cp bin/avalanchemq /usr/sbin/avalanchemq
```

## Usage

`avalanchemq -D /var/lib/avalanchemq`
